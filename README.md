# ML Playground

  ***This is currently a work in progress. Changes are being made every day***

Welcome to **ML Playground**, a hands-on machine learning lab built from scratch—no libraries, just logic.  
This repository features core ML algorithms like Neural Networks, Decision Trees, Logistic Regression, and more, implemented entirely from the ground up to demystify how they actually work under the hood.

---

## 🚀 Overview

ML Playground is designed to provide a clear and intuitive understanding of machine learning algorithms by implementing them from scratch.  
This approach allows you to see the inner workings of each algorithm and gain a deeper appreciation for their mechanics.

---

## 🧠 Key Features

- **Modular Python Implementations**: Each algorithm is implemented as a standalone module, making it easy to understand and modify.
- **Flask Web Application**: A user-friendly interface to visualize and interact with the models.
- **Interactive Components**: Visualizations and interactive elements to help you grasp complex concepts.
- **Extensive Documentation**: Clear explanations and comments to guide you through the code.

---

## 📂 Project Structure

- `models/`: Contains the core machine learning algorithms implemented from scratch.
- `frontend/`: Houses the Flask web application for visualizing and interacting with the models.
- `notebooks/`: Jupyter notebooks for experimentation and testing.
- `tests/`: Unit tests to ensure the correctness of the implementations.
- `utils/`: Utility functions and helper scripts.
- `data/`: Contains raw and processed datasets for testing and training the models.
- `requirements.txt`: Lists the dependencies required to run the project.
- `main.py`: The entry point for running the Flask application.
- `notes.txt`: Contains additional notes and observations.

## 🚀 Getting Started

### 🔧 Prerequisites

- Python 3.7+
- pip

### 🛠️ Installation

```bash
# Clone the repo
git clone https://github.com/Pineapsle/ML_Playground.git
cd ML_Playground

# (Optional) Create a virtual environment
python -m venv venv
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
